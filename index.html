Predicting Sales Using Machine Learning

For this analysis, I was the team leader and we used data of video game sales from North America and Japan to see if a Linear Regression Model would predict if European sales would be greater than 75% (threshold). The data was explored, and we kept platform, genre, and sales from North America, Japan, and Europe. Three databases were created in PgAdmin and stored in Heroku, a cloud-based storage system. The databases were combined using Python coding which was then pushed into a Machine Learning Model called Linear Regression resulting with an 88% accuracy showing that 75% or above sales were successful in Europe; thus concluding that launching a video game in Europe could be successful.
